# Chaewon Heo Portfolio

GitHub Pages용 정적 포트폴리오 사이트 템플릿입니다.

## 파일 구성

- `index.html`: 메인 페이지
- `styles.css`: 전체 스타일

## GitHub Pages 배포 방식

### 1. 개인 사이트로 배포

저장소 이름을 `heorshey.github.io`로 만들면 주소는 아래처럼 됩니다.

- `https://heorshey.github.io`

이 경우 현재 파일들을 저장소 루트에 올리면 됩니다.

### 2. 프로젝트 사이트로 배포

저장소 이름을 `chaewonheo`로 만들면 주소는 아래처럼 됩니다.

- `https://heorshey.github.io/chaewonheo`

현재 템플릿은 상대 경로를 사용하므로 이 방식도 그대로 동작합니다.
